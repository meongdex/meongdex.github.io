Pasang Meongdex di Android

Dua cara mudah: pasang langsung dari browser (paling cepat), atau hasilkan file APK lewat PWABuilder.

Cara 1 Pasang langsung dari Chrome (paling cepat)

Tidak perlu file APK. Meongdex adalah PWA — bisa langsung dipasang jadi aplikasi dari browser.

1
Buka Meongdex game di Chrome Android.
2
Tekan menu titik tiga di pojok kanan atas Chrome.
3
Pilih Instal aplikasi (atau "Tambahkan ke Layar Utama").
4
Meongdex muncul sebagai ikon di layar utama, jalan fullscreen tanpa address bar.
Buka Meongdex game

Cara 2 Hasilkan file APK via PWABuilder

Mau file .apk untuk dibagikan atau dipasang manual? Pakai PWABuilder (gratis, tanpa kartu kredit).

1
Buka pwabuilder.com di browser komputer.
2
Tempel URL game Meongdex di kotak input, lalu tekan Start.
3
Tunggu PWABuilder menganalisis. Pastikan skor manifest & service worker hijau (Meongdex sudah punya keduanya).
4
Pilih paket Android, tekan Generate, isi nama aplikasi "Meongdex".
5
Unduh file meongdex.apk yang dihasilkan.
6
Pindahkan ke HP Android, izinkan sumber tidak dikenal di Pengaturan, lalu ketuk file APK untuk dipasang.

URL game untuk ditempel di PWABuilder:

https://meongdex.github.io/game/

Meongdex sudah memenuhi syarat PWA installable: manifest lengkap (name, icons 192/512/maskable, start_url, display standalone, theme_color), service worker terdaftar, dan dilayani via HTTPS dari GitHub Pages.

Catatan soal file APK siap pakai

File meongdex.apk siap unduh akan ditambahkan ke halaman ini setelah dibangun lewat PWABuilder. Untuk sekarang, gunakan Cara 1 (pasang dari Chrome) yang jalan langsung tanpa APK — atau Cara 2 untuk menghasilkan APK sendiri dalam sekitar 2 menit.
