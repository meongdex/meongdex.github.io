Kebijakan Privasi Meongdex

Terakhir diperbarui: 4 Juli 2026

Meongdex dibuat karena sayang sama kucing-kucing di sekitar kita. Privasi kamu adalah prioritas utama proyek ini. Kebijakan ini menjelaskan data apa yang dikumpulkan, bagaimana dipakai, dan kontrol yang kamu punya.

1. Data yang Tersimpan Lokal di Perangkatmu

Semua data tersimpan lokal di perangkatmu (IndexedDB + localStorage browser). Tidak ada yang dikirim ke server mana pun.

  • Foto kucing — dikompres dan dibersihkan dari metadata EXIF (termasuk data GPS) sebelum disimpan, murni untuk performa dan privasi.
  • Metadata kucing — nama, warna, kelangkaan, temperamen, tanggal, koordinat lokasi (opsional, bisa dilewati).
  • Progres pemain — XP, level, streak, misi, achievement, pengaturan.
  • Lokasi — hanya disimpan kalau kamu izinkan Geolocation API. Bisa dilewati dengan "Lanjut tanpa lokasi".

2. Pindah Device (Import/Export Manual)

Meongdex tidak menggunakan sistem akun atau server sync. Untuk pindah device, gunakan fitur Ekspor Meongdex (unduh file JSON berisi foto + metadata) di Pengaturan, lalu Impor Meongdex di device baru. File cadangan ini berisi semua data koleksi kamu — simpan di tempat aman.

Leaderboard (opsional, butuh konfigurasi developer)

Fitur leaderboard (papan peringkat) opsional dan butuh konfigurasi Supabase oleh developer. Kalau aktif, hanya nama panggilan + total XP + jumlah kucing yang dikirim — TIDAK PERNAH foto atau lokasi. Kalau belum dikonfigurasi, fitur ini menampilkan pesan ramah dan tidak mengganggu gameplay.

3. Verifikasi AI (COCO-SSD)

Verifikasi foto kucing dijalankan di browser kamu menggunakan TensorFlow.js + COCO-SSD. Foto tidak diunggah ke server untuk verifikasi — model AI dimuat sekali via CDN lalu berjalan lokal.

4. Layanan Eksternal

  • Open-Meteo API — cuaca, berdasarkan koordinat lokasi. Tidak mengirim data identitas.
  • OpenStreetMap (Leaflet tiles) — peta hotspot. Hanya mengirim koordinat untuk render tile.
  • Google Fonts — tipografi (Fredoka, Plus Jakarta Sans, JetBrains Mono).
  • CDN: jsDelivr — library TensorFlow.js, COCO-SSD, idb, Leaflet.
  • Supabase (opsional) — leaderboard saja (nama panggilan + XP + jumlah kucing).

5. Cookie dan Storage

Meongdex tidak menggunakan cookie tracking. Penyimpanan lokal yang dipakai:

  • localStorage — preferensi pemain, progres.
  • IndexedDB — foto + metadata kucing (database lokal).
  • Cache Storage — service worker cache untuk offline mode.

Semua bisa dihapus kapan saja via Pengaturan > Hapus semua data, atau clear browser cache.

6. Hak Kamu

  • Akses — lihat semua data via Pengaturan > Ekspor Meongdex.
  • Portabilitas — export ke JSON, import di device lain.
  • Penghapusan — Pengaturan > Hapus semua data, atau clear browser cache.
  • Offline — game tetap bisa dimainkan tanpa internet setelah kunjungan pertama.

7. Anak di Bawah Umur

Meongdex tidak ditujukan untuk anak di bawah 13 tahun. Tidak ada data yang sengaja dikumpulkan dari anak-anak. Kalau kamu orang tua/wali dan tahu anak kamu mengirim data tanpa izin, hubungi developer untuk penghapusan.

8. Keamanan

Data lokal di perangkatmu aman selama perangkat tidak diakses orang lain. Kalau leaderboard aktif, data dikirim ke Supabase via HTTPS (enkripsi in-transit).

9. Perubahan Kebijakan

Kebijakan ini bisa berubah kalau ada fitur baru. Perubahan akan diumumkan di Progress Log README. Tanggal "Terakhir diperbarui" di atas akan diupdate.
