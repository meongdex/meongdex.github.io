Install APK

Pasang Meongdex di Android

Mau pasang seperti aplikasi biasa? Unduh APK-nya lalu ikuti empat langkah berikut. Untuk pengguna iPhone atau cukup main di browser, langsung tekan Main di Browser — Meongdex bisa dipasang ke layar utama tanpa APK.

1

Unduh file APK

Tekan tombol Download APK Android di atas. File meongdex.apk akan tersimpan di folder Download HP-mu.

2

Izinkan sumber tidak dikenal

Buka Pengaturan Android → Keamanan → aktifkan "Install unknown apps" untuk browser yang kamu pakai mengunduh.

3

Buka file APK

Buka folder Download, ketuk meongdex.apk. Android akan menampilkan konfirmasi instalasi.

4

Install & main

Tekan Install, tunggu sebentar, lalu buka. Meongdex jalan sama persis seperti versi browser.

FAQ

Pertanyaan yang sering muncul

Apakah Meongdex benar-benar gratis?
Ya. Tidak ada biaya, tidak ada langganan, tidak ada iklan. Semua library dan layanan yang dipakai (TensorFlow.js, IndexedDB, service worker) gratis dan berjalan di perangkatmu sendiri.
Apakah foto kucing saya dikirim ke server?
Tidak. Foto dan lokasi hanya disimpan lokal di perangkatmu (IndexedDB). Verifikasi AI juga dijalankan di browsermu, bukan di cloud. Tidak ada data yang diunggah ke mana pun.
Kenapa harus izinkan sumber tidak dikenal untuk APK?
Karena APK Meongdex tidak didistribusikan lewat Play Store. Android meminta izin ini sebagai pengaman agar kamu sadar sedang memasang aplikasi dari luar Play Store. Setelah terpasang, kamu bisa mematikan kembali izin tersebut.
Bagaimana jika AI tidak mengenali kucing di foto saya?
Tidak masalah. Meongdex tidak memblokir kamu. Kalau AI belum yakin, muncul tombol "Ya, ini kucing" untuk konfirmasi manual. Kamu tetap bisa menyimpan kartunya.
Bisa main tanpa internet?
Setelah kunjungan pertama, shell aplikasi dan Meongdex-mu tetap bisa dibuka tanpa internet berkat service worker. Deteksi AI butuh internet sekali di awal untuk mengunduh model, lalu tersimpan untuk pemakaian berikutnya.

Siap ketemu kucing pertamamu?

Keluar rumah, cari kucing di sekitar, kasih makan, dan abadikan. Meongdex-mu menunggu kartu pertama.
